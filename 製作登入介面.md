#### 管理 Mongoose Models
>- [[用資料夾管理 Mongoose models 模組]]
>- [[Validators]]
建立 models -> user.js
```js
// import
const mongoose = require('mongoose');
// define a schema
const userSchema = new mongoose.Schema({
	username: {
		type: String,
	},
	password: {
		type: String,
	}
})

// create a model
const User = mongoose.model('User', userSchema);
// export(到 app.js 用)
module.exports = User;
```
