## useNavigate Hook(useHistory)
>[stackoverflow](https://stackoverflow.com/questions/64838587/how-to-properly-use-usehistory-from-react-router-dom)
### v6
```javascript
const navigate = useNavigate()
navigate('/')
```
### 舊版本差異
```javascript
const history = useHistory()
history.push('/')
```
