## 物件 Object
- 屬性(properties) 的集合
- 屬性是一個：[Key-Value Pair](Key-Value%20Pair.md) 
	- 值 value 可以是任何[資料型別](資料型別.md)
		- 值是 `function` 的時候則叫做 `method`
	- 另外在 ES6 有規範了 [Method 簡寫語法糖](Method%20簡寫語法糖.md)
- [Function](Function.md), [Array](Array.md) 都是一種 Object


```js
let home = {
	name : 'Mary',
	key : value,
	properties,
	method,
}
```
