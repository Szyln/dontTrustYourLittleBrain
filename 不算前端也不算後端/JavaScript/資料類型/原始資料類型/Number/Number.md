# Number
- NaN: not a number
- Infinity: 無限大
- [Floating Point](Floating%20Point.md)

```js
console.log(5 / 0); // log Infinity
console.log(Infinity / Infinity); // log Infinity 跟微積分有關（好）
NaN === NaN; // log false 想要認定是否為 NaN 要用 isNaN
```
