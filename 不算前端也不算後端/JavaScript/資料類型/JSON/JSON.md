# JSON
- JavaScript Object Notation
- 並不屬於 JS，但 JS 有很多功能可以轉換 JSON 成 JS 讀得懂的功能
- 跟 object 寫法很像，不過 key 的部份要用 ”” 包起來
- 最外邊是 `[]` 或 `{}` 都有可能
- 前端工程師常常撈特定資料存在一個變數裡
- 後端 Database 有可能會用到類似的
