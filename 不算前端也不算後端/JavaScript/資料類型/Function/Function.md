## 函式 function
```js
// function declaration
function name(parameter) {
	// code 
	// parameter 可以放進來寫
}

// invoke(執行)
名稱(argument);
```

- 寫程式的時候，會遇到好幾句程式碼才能完成一個動作的時候，這時候可以製作 function
- 將繁複的程式碼整理成一個功能，增加
	- 重複利用性
	- 可讀性
	- 方便使用性
- 再設定「參數」，讓 function 可以更加客製化
