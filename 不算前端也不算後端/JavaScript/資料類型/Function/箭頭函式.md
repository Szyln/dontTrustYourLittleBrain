## 箭頭函式
```js
( (parameter) => {<自帶 return>})
```
```js
(parameter) => (<return 的值，不用再寫 return>)
```

- `()`： parameter 若只有一個的話，可省略
- `{}`： 只有一行的話，可省略
- 自帶 `return`
- 本身無自己的 `this` 指向，看外層



### this 指向
>- [this 的指向](this%20的指向.md)
>- [Function Execution Context](Function%20Execution%20Context.md)：箭頭函式不形成 `this`

- 沒有自己的 this
- 看上層 function 的 this 是誰
- 上層不是 function 就是全域
