## Falsy Value
- [Boolean](Boolean.md) 的 `false`
- [Null](Null.md)
- [undefined](undefined.md)
- [Number](Number.md) 的 `0`, `NaN`(not a number)
- [String](String.md) 的 `''`

### 不是 Falsy 
- 具記憶體的[資料型別](資料型別.md)
	- [Object](Object.md)（但 [Null](Null.md) 是 [Falsy Value](Falsy%20Value.md)）
	- [Array](Array.md)
