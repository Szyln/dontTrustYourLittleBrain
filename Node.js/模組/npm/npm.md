# npm(node package manager)
管理並追蹤他人撰寫的 Node [[模組]]，安裝 Node.js 就會一起下載下來囉！

## [[初始化(package.json)]]
當專案要做 npm 的使用時，都必須先初始化環境，製作一個 `package.json` 的檔案，有這個檔案才能上 npm 去使用別人的模組
```
// 專案資料夾
npm init
```
