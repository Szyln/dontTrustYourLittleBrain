# connect-flash
[npm](https://www.npmjs.com/package/connect-flash)
使用 flash 需要安裝的 npm 模組
```
$ npm install connect-flash
```

## 使用
```js
const flash = require('connect-flash');
const app = express();

// middleware: 會生成一個 req.flash 可做使用
app.use(flash());	
```
