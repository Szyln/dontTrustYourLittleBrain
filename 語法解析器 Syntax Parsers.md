## 語法解析器 Syntax Parsers
- 將你寫的程式碼 --（如果你的程式碼是有效的）--> 電腦看得懂的程式
- 他可能是一個直譯器或編譯器

```mermaid
graph TB
A([你寫的程式碼]) --> B{程式碼是否有效} 
B -.->|是|C[或許會作些其他事]
C -->D[轉換成電腦看得懂的內容]
B -->|否|E[跳出程式碼無效的提示]
```


### 一些小提醒
return 後面一定要接東西，不然語法解析器會自動加 `;`
```js
function fn1() {
	return 
	// 不要直接換行
	(
		2 + 1
	)
}
```
