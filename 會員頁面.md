# 會員頁面
- 是否已經登入？（session 內是否有 isVerified）
	- 否：導回登入頁面
	- 是：顯示

>- [[Routing 中間使用 Middleware]]
>- [[Session]]
```js
const requireLogin = (req, res, next) => {
	if(!req.session.isVerified == true) {
		res.redirect('login');
	} else {
		next();
	}
}

```
```js
app.get('/member', requireLogin, (req, res) => {
	res.render('member');
})
```
