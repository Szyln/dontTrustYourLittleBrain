## 設定位置起始點隨機
>gsap 還有其他可以簡略的數學計算寫法：

gsap 的簡略寫法

```js
gsap.from('.circle', {
	duration: 1,
	opacity: 0,
	y: 'random(-200, 200)'	// 起始的 y 軸隨機(正負200)
	stagger: 0.25
})
```
### 原始寫法
```js
gsap.from('.circle', {
	duration: 1,
	opacity: 0,
	y: () => Math.random() * 400 - 200,	// 起始的y軸隨機
	stagger: 0.25
})
```
