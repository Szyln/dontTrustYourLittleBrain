# nodemon
[nodemon](https://www.npmjs.com/package/nodemon)
code 更新後， cmd 會自動重新執行 node.js

## 安裝
電腦全域安裝（推薦）
```shell
npm install -g nodemon
```
專案安裝
```shell
npm install --save-dev nodemon
```

## 使用
```shell
nodemon app.js	// 執行專案的 js 檔案
```
