## 重複宣告一個外層的變數
- 重複宣告一個外層的變數（限使用之前），會以該層為準

> 該層的重複宣告若是在使用之後，就會 TDZ

```js
// 外面宣告 const, let, var 都可，function 之上下都可
// const d = 1
// let d
// var d
function fn() {
  // const d	= 1		 // 1
  // let d			 // undefined
  // var d			 // undefined
  console.log(d)
	// const d	= 1		 // TDZ 炸
  // let d			 // TDZ 炸
  // var d			 // undefined, hoisting
}
// const d = 1
// let d = 1
// var d

fn()
```
